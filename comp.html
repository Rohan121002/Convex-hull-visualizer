Convex Hull Algorithms Comparision

Jarvis March Algorithm

Time Complexity: O(n*h)

Space Complexity: O(h)

Advantages:

  • Simple implementation
  • Works well for small or moderate datasets

Disadvantages:

  • Inefficient for large datasets
  • Sensitive to input ordering

Kirkpatrick–Seidel Algorithm

Time Complexity: O(nlog h)

Space Complexity: Depends on data structures

Advantages:

  • Efficient for large datasets
  • Handles dynamic datasets well

Disadvantages:

  • More complex implementation
  • Requires additional data structures

Comparison:

  • Efficiency: Kirkpatrick–Seidel is more efficient for large datasets.
  • Space Usage: Jarvis March has a fixed space complexity, while Kirkpatrick–Seidel's space usage depends on data structures.
  • Implementation Complexity: Jarvis March is simpler to implement, whereas Kirkpatrick–Seidel requires more complexity.
  • Robustness: Kirkpatrick–Seidel is more robust for large or dynamic datasets.

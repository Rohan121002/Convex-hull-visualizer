Jarvis March Algorithm Analysis

Overview:

The Jarvis March algorithm, also known as the Gift Wrapping algorithm, is used to compute the convex hull of a set of points in the plane. It iteratively selects points that form the convex hull by wrapping a "gift" around the points.

Algorithm Steps:

  1. Select the point with the lowest y-coordinate (and the leftmost if there are ties) as the starting point.
  2. Initialize an empty list to store the convex hull points.
  3. Repeat the following until the current point is the starting point again:
    1. Initialize a variable to store the next point on the hull as the point with the smallest polar angle from the current point.
    2. Add the next point to the convex hull list.
    3. Update the current point to be the next point.

Implementation Considerations:

When implementing the Jarvis March algorithm, several considerations should be kept in mind:

  • Input Order: The order in which points are processed can affect the algorithm's performance. Sorting the points by their polar angles from the current point can improve efficiency.
  • Collinear Points: Collinear points can lead to incorrect results. Handling collinear points by removing duplicates or using epsilon values for comparisons is necessary.
  • Efficiency: While the algorithm is simple, its worst-case time complexity can be \( O(n^2) \) for \( n \) input points. Implementing optimizations like Graham's scan can improve efficiency.
  • Convex Hull Orientation: Ensure that the convex hull points are returned in counterclockwise order or clockwise order, depending on the application's requirements.
